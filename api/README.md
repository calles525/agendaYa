# Booking SaaS - Sistema de Gestión de Citas y Alquileres

## 📋 Descripción
Sistema SaaS multi-usuario para gestión de citas y alquileres. Permite a proveedores ofrecer servicios de citas (consultas, asesorías) y alquiler de productos por hora.

## 🚀 Características
- ✅ Multi-rol: Admin, Proveedores (negocios/individuales), Clientes
- ✅ Módulo de citas con especialistas y especialidades
- ✅ Módulo de alquiler de productos por hora
- ✅ Sistema de reservas con múltiples estados
- ✅ Historial clínico/de servicio por cliente
- ✅ Notificaciones por Telegram
- ✅ Calificaciones y reseñas
- ✅ Dashboard con reportes
- ✅ Búsqueda avanzada con filtros

## 🛠 Tecnologías
- **Backend:** Node.js + Express
- **Base de datos:** MySQL + Sequelize ORM
- **Autenticación:** JWT + bcrypt
- **Notificaciones:** Telegram Bot API
- **Validaciones:** express-validator
- **Logs:** Winston
